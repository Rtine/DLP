Vous devez télécharger deux logiciel : Arduino et Processing

Dans le dossier Drone : 

-le .ino correspond à la fonction principale du programme, elle permet d'apporter les insrtructions des touches de la console
-les .h sont quant à elle les fonctions de différents composant, ils doivent être impérativement dans le même dossier que DLP.ino


Dans le dossier LidarScanner : 

-le .ino correspond au programme du LIDAR


Dans le dossier LidarScanner1 : 

-le .pde correspond au programme sous processing, ce qui nous permettra d'afficher la console pour le nuage de point du LIDAR
-le programme du dossier LidarScanner, LidarScanner.ino, doit être également lancer, afin que celui-ci puisse récupérer les donnés



Étant donner que vous n'avez pas les composants, la vérification ainsi que le téléversement (si jamais vous posséder un Arduino) suffit.
